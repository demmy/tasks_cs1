Создать отдельный Solution в папочке MainPatternsВашеИмя.
Создать два подпроекта (один - для MVP, второй - для MVVM).

Создать WinForm приложения в каждом подпроекте. Выглядеть и работать они должны одинаково, но должны быть построены на двух разных шаблонах.

Приложение должно состоять из одной формы, в которой будут:

- комбобокс или радиокнопка с вариантами операции (сложение, вычитание, умножение, деление)
- два edit, в которые будут вводиться числа
- кнопка "выполнить", по которой будет выполняться операция.
- ещё один edit, который невозможно редактировать. В нём будет отображаться результат выполнения операции.
